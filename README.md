# Shaanvik Group - Luxury Jewelry Store

![Shaanvik Group](https://images.unsplash.com/photo-1515562141207-7a88fb7ce338?q=80&w=1600&auto=format&fit=crop&w=1200&h=400)

A premium, responsive website for Shaanvik Group, Bangkok's premier destination for bespoke luxury jewelry. This modern web application showcases custom engagement rings, fine necklaces, designer earrings, and expert jewelry repair services.

## ✨ Features

### 🏠 **Home Page**
- Hero section with compelling brand messaging
- Featured services showcase (Custom Rings, Necklaces, Earrings, Repairs)
- Why choose us section with key differentiators
- Process overview with 4-step visualization
- Customer testimonials with authentic reviews
- Multiple call-to-action sections

### 📖 **About Page**
- Company story and founding narrative
- Core values (Craftsmanship, Quality, Creativity, Personalization)
- Team member profiles with photos and bios
- Process preview linking to detailed process page
- Statistics and achievements display

### ⚙️ **Process Page**
- Detailed 4-step bespoke jewelry creation process
- Visual process cards with icons and descriptions
- Client journey from inspiration to final delivery

### 🖼️ **Gallery Page**
- Jewelry collection showcase
- Interactive image gallery with lightbox functionality
- Category-based organization

### 📞 **Contact Page**
- Contact information and location details
- Professional inquiry form with validation
- Business hours and contact methods

## 🚀 Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/shaanvik-group.git
   cd shaanvik-group
   ```

2. **Start local server**
   ```bash
   # Using Python (recommended)
   python3 -m http.server 8000

   # Or using Node.js
   npx serve .

   # Or using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

## 🔧 Customization

### Colors
The design uses CSS custom properties for easy theming:
```css
:root {
  --gold: #C5A059;
  --pastel-blue: #E3EBF2;
  --dark: #1A1A1A;
  --white: #FFFFFF;
}
```

### Content
- Update text content in respective HTML files
- Replace images in `src` attributes
- Modify contact information in footer and contact page

### Styling
- Main styles in `style.css`
- Component-specific styles organized by page
- Responsive breakpoints at 768px and 480px
